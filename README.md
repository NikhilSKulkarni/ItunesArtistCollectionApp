# ItunesArtistCollectionApp
ItunesArtisArtist Application demonstrate the collections of the artist by using itunes API.
**Features**
1. **MVVM Design Pattern:** Separates business logic from UI components, enhancing code maintainability and testability.
2. **Swift Language:** Developed using modern Swift features and best practices.

**Purpose**
This poroject purpose is to build an application using MVVM architecture and implement network call and show the received data on screen.
This porject also demonstrate how to add async images and displays.
